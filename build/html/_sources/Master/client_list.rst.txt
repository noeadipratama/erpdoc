Client List
=============

Anda dapat menambahkan daftar client Anda sendiri ke situs web. Untuk menambahkan atau melihat daftar client, Anda bisa mengklik menu 'Master' dan pilih 'client'.

.. image:: ../img_src/dashboard_master.png
    :width: 180px
    :alt: Login Section

.. image:: ../img_src/client_list2.png
    :width: 700px
    :alt: Login Section

- Ada daftar client yang akan ditampilkan di halaman ini. Ini akan menunjukkan options, nama client, active dari client tersebut.
- Anda juga bisa menyortir client dengan nama client atau active menurut abjad.

Create New Client
-------------------

 .. image:: ../img_src/client_add_button.png
     :width: 200px
     :alt: Login Section

 - Klik tombol "Add client" di kiri atas halaman ini.
 
 .. image:: ../img_src/client_add.png
    :width: 400px
    :alt: Login Section
 
 - Muncul Form Add dan Anda bisa memasukkan Client Name, Client Address, Client Phone, Client Email dan Active.
 - Kemudian klik tombol "Save" untuk menyimpan data client. 
 
Edit/Update Client
--------------------
  .. image:: ../img_src/client_option.png
    :width: 150px
    :alt: Login Section

 - Anda bisa klik tombol "Edit" pada baris yang ingin Anda edit.

 .. image:: ../img_src/client_edit.png
    :width: 600px
    :alt: Login Section
 
 - Form Edit akan dibuka dan Anda bisa mengubah data client di sana.
 - Kemudian klik tombol "Save" untuk menyimpan data client.
