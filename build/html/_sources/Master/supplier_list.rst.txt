supplier List
=============

Anda dapat menambahkan daftar supplier Anda sendiri ke situs web. Untuk menambahkan atau melihat daftar supplier, Anda bisa mengklik menu 'Master' dan pilih 'supplier'.

.. image:: ../img_src/dashboard_master.png
    :width: 180px
    :alt: Login Section

.. image:: ../img_src/supplier.png
    :width: 700px
    :alt: Login Section

- Ada daftar supplier yang akan ditampilkan di halaman ini. Ini akan menunjukkan options, nama supplier, alamat supplier, telp supplier, email supplier, active dari supplier tersebut.
- Anda juga bisa menyortir supplier dengan nama supplier atau active menurut abjad.

Create New supplier
-------------------

 .. image:: ../img_src/supplier_add_button.png
     :width: 200px
     :alt: Login Section

 - Klik tombol "Add supplier" di kiri atas halaman ini.
 
 .. image:: ../img_src/supplier_add.png
    :width: 400px
    :alt: Login Section
 
 - Muncul Form Add dan Anda bisa memasukkan Nama supplier dan Active.
 - Kemudian klik tombol "Save" untuk menyimpan data supplier. 
 
 
Rate supplier
-------------------

 .. image:: ../img_src/supplier_option.png
     :width: 200px
     :alt: Login Section

 - Klik tombol "Rate" pada baris yang Anda inginkan.
 
 .. image:: ../img_src/supplier_rate.png
    :width: 400px
    :alt: Login Section
 
 - Muncul data rating supplier.
 
Edit/Update supplier
--------------------
  .. image:: ../img_src/supplier_option.png
    :width: 150px
    :alt: Login Section

 - Anda bisa klik tombol "Edit" pada baris yang ingin Anda edit.

 .. image:: ../img_src/supplier_edit.png
    :width: 600px
    :alt: Login Section
 
 - Form Edit akan dibuka dan Anda bisa mengubah data supplier di sana.
 - Kemudian klik tombol "Save" untuk menyimpan data supplier.
