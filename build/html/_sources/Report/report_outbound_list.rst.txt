Report Outbound List
=============

Anda dapat menambahkan daftar sales order Anda sendiri ke situs web. Untuk menambahkan atau melihat daftar sales order, Anda bisa mengklik menu 'Report' dan pilih 'Report Outbound'.

.. image:: ../img_src/report_outbond.png
    :width: 800px
    :alt: Login Section

- Ada Form input data berupa tanggal, isi form input Start berupa tanggal awal dan form input End berupa tanggal akhir.
- Klik button 'Export' akan mengekpor file berupa excel.
