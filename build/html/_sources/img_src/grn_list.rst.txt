Good Receive Note List
=============

Anda dapat menambahkan daftar inbound Anda sendiri ke situs web. Untuk menambahkan atau melihat daftar inbound, Anda bisa mengklik menu 'Master' dan pilih 'Good Receive Note'.

.. image:: ../img_src/grn.png
    :width: 800px
    :alt: Login Section

- Ada daftar inbound yang akan ditampilkan di halaman ini. Ini akan menunjukkan options, inbound ID, Supplier, Ext No., Order ID, Warehouse, Cdate dan Cuser dari inbound tersebut.
- Anda juga bisa menyortir inbound dengan inbound ID atau active menurut abjad.

Good Receive Note Detail
-------------------

 .. image:: ../img_src/inbound_add_button.png
     :width: 200px
     :alt: Login Section

 - Klik tombol "Create Good Receive Note" di kiri atas halaman ini.
 
 .. image:: ../img_src/grn_detail.png
    :width: 400px
    :alt: Login Section
 
 - Muncul Form Add dan Anda bisa memasukkan Ext No., Order ID, Supplier dan Warehouse.
 - Kemudian klik tombol "Save" untuk menyimpan data inbound. 
 
Edit/Update Good Receive Note
--------------------
  .. image:: ../img_src/inbound_option.png
    :width: 150px
    :alt: Login Section

 - Anda bisa klik tombol "Edit" pada baris yang ingin Anda edit.

 .. image:: ../img_src/inbound_edit.png
    :width: 600px
    :alt: Login Section
 
 - Form Edit akan dibuka dan Anda bisa mengubah data inbound di sana.
 - Kemudian klik tombol "Save" untuk menyimpan data inbound.
